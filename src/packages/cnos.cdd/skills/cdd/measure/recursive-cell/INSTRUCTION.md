# cnos recursive-cell semantic witness instruction v0.1.0

Apply TSC Core 3.2 and the v3.2.4 fixed defect checklist to the provided
ordered bundle. Return the standard TSC witness JSON contract, with the exact
target name and no computed Coh/C-sigma fields.

Additionally, the evidence and defect cards must cover every applicable level:

- L0 three-cell properties, relations, and system process;
- L1 internal CCNF components, signatures, firebreaks, and recursion;
- L2 contract/CM/evidence/receipt/V/delta types and migrations;
- L3 cell FSM and wave FSM under alpha/beta/gamma;
- L4 actual matter, receipts, roles, review bytes, and state history.

For FSMs, do not score a state-name sequence as an executable FSM. Check state
closure, totality, determinism/priority, reachability, authority uniqueness,
guard fixtures, command/table parity, repair/recovery, terminal semantics, and
CC-disposition mapping.

Apply the primary-axis precedence from TSC v3.2.4. Mark secondary axes without
double filing. Use `confidence <= 0.75` if any required source level is absent.

After per-level scoring, check all thirteen hard invariants in SKILL.md §7.
Any failed hard invariant must appear as a systemic defect and must prevent an
accepting ratification disposition, even if a geometric aggregate exceeds the
threshold.

The CM or prompt used for measurement must be immutable and selected before
the judgment. A CM derived from the target may be emitted as a future candidate
artifact, but it cannot provide independent warrant for the measurement that
created it.
