## What was hard

- **Friction note 1 (claim guard framing) was the one place γ's scaffold explicitly declined to pre-decide, and it was the hardest call in the cycle.** Neither of γ's two offered framings (a new run-id-comparing fact, or dropping the run-based guard structurally) was the eventual answer — the resolution was a hybrid neither option anticipated: reuse the existing `run_active` guard unmodified (its branch-scoped definition already structurally excludes the requesting wake's own pre-branch execution, since that run executes on `main`, not on a `cycle/{issue}` branch that doesn't exist yet at claim time) plus a new `claim_request_present` marker guard. Getting to that hybrid required running the TDD sanity check (checking out pre-#575 `transitions.json` and confirming which of the 14 new tests fail) to prove the marker-file half was load-bearing and not redundant with the run-check half — a case where the empirical check, not the design reasoning alone, was what closed the ambiguity with confidence.
- **Rule ordering (friction note 6) required care because `transitions.json`'s first-match-wins semantics meant three new `in-progress` rules had to be threaded between the existing review-request rules and the existing dead-run reconciliation rules without shadowing either.** The dedicated non-shadowing test plus a full-suite unmodified-assertion re-run were both necessary; either alone would have left a gap (the dedicated test only proves the three named pre-existing fixtures; the full suite alone would not have named *which* fixtures mattered if something had broken).
- **Distinguishing what belonged in this cycle's surface list from what looked like the same pattern one layer up.** `delta/SKILL.md` §9.6 carries the same "direct write" prose shape as the sites this cycle did fix, but describes δ *signaling* a needed transition, not δ *performing* the write — a distinction that took deliberate re-reading to hold onto rather than either reflexively fixing it (scope creep) or reflexively ignoring it (silent gap). The chosen resolution — read it in full, leave it unedited, record the reasoning as debt — is what let γ triage it cleanly into a follow-up (#589) after the fact.
